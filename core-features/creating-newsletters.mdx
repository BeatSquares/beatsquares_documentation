## What You'll Learn

In this guide, you'll learn how to:
- Create a newsletter with custom content segments
- Add articles to newsletter segments
- Generate and review newsletter variants
- Edit and verify newsletter content
- Publish and export your newsletter

## Before You Start

**Prerequisites:**
- Active BeatSquares account with newsletter access
- At least one Square created with the "newsletter" output format

<Note>
Each square has a specific output format. To create a newsletter, you need a square with the output format "newsletter".
</Note>

**What You'll Need:**
- Newsletter content and messaging strategy
- Articles or content sources to include
- Brand assets (logos, colors, fonts) if customizing

## Editing Content Segments

Newsletters are built using segments. Each segment can contain custom text content or articles, events or press releases. The segments are pre-defined per Square as part of the so called "Blueprint". If you want to change the segments, please contact BeatSquares. You'll edit segments one at a time.
