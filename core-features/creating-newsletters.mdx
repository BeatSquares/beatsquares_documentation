## What You'll Learn

In this guide, you'll learn how to:
- Create newsletter templates and layouts
- Add content and customize design elements
- Configure distribution lists and settings
- Schedule and send newsletters to subscribers

## Before You Start

**Prerequisites:**
- Active BeatSquares account with newsletter access
- At least one Square created for organization

**What You'll Need:**
- Newsletter content and messaging strategy
- Subscriber list or lead magnets for list building
- Brand assets (logos, colors, fonts)
- Content calendar or publishing schedule

## Create a newsletter
Go to the square where you want to create the newsletter. 

**Please note:** Each square has a specific output format. So to create a newsletter, you need a square with the output format "newsletter".

## Redoing the newsletter
If you are not happy with the generated newsletter or would like to substantially alter the content for a different reason, you can simply "Redo" it.


### 1. When you click on "Redo" you can do two things: Segment Creation or Content Review

You can either go back to the beginning and edit the segments (change to different articles, add custom prompts etc.) or you can review the generated content to manually change the newsletter. The latter is useful for minor tweaks, whereas the former allows you to fundamentally change the newsletter.  
These features work as described above.
