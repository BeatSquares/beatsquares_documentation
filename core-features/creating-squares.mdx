## What You'll Learn

In this guide, you'll learn how to:

- Create a new Square from scratch
- Select an output format (Podcast, Newsletter, WhatsApp)
- Choose a data source for your content
- Review and manage your Square settings

## Before You Start

**Prerequisites:**
- An admin account for BeatSquares
- Understanding of what Squares are and their purpose
- Access to the BeatSquares dashboard

**What You'll Need:**

- A clear name for your Square
- Decision on output format (Podcast, Newsletter, or WhatsApp)
- Knowledge of which data source contains your content

## Creating Squares

Login to your BeatSquares account and navigate to the [Dashboard](https://app.beatsquares.com/).

### 4. Select a data source

Click on the data source you want to use for this Square. The available data sources are configured by your organization. Contact BeatSquares support if you need additional data sources.

<Note>
Data sources determine which content will be available when creating your media items. Each data source represents a specific content collection configured for your organization.
</Note>

## Best Practices

- **Descriptive Names:** Use clear, specific names that explain the Square's purpose
- **Choose the Right Format:** Select the format (Podcast, Newsletter, or WhatsApp) that best matches your content distribution strategy
- **Understand Your Data Source:** Make sure you select the data source that contains the content you need
- **Test Before Committing:** Remember that you cannot change the format or data source after creation
