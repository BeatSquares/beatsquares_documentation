## Account Setup

Account setup for BeatSquares is handled directly by our team. We manage user accounts to ensure proper access control and provide personalized onboarding support.

## Getting an Account

If you'd like to create a BeatSquares account, please reach out to us:

- **Email**: [support@beatsquares.com](mailto:support@beatsquares.com)

We'll work with you to set up your account and get you started with creating squares, podcasts, newsletters, and WhatsApp messages.
